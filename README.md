Abyaz Ahmed’s Portfolio Website

Welcome to the repository for my personal portfolio website! This website is a showcase of my projects, skills, and contact information, designed to present my work in a visually appealing and interactive way.

Features

	•	Interactive Design: Includes animations, hover effects, and a custom cursor for enhanced user experience.
	•	Dark/Light Mode: Theme toggle button for switching between light and dark modes.
	•	Dynamic Portfolio: Highlights my key projects and provides links to my GitHub profile for more details.
	•	Responsive Layout: Optimized for all devices, from desktops to mobile screens.
	•	Contact Section: Easy-to-use contact options including email, LinkedIn, and WhatsApp.

Technologies Used

	•	HTML5: For structuring the content.
	•	CSS3: For styling and animations.
	•	JavaScript: For interactivity, smooth scrolling, and theme toggle functionality.
	•	AOS (Animate On Scroll): For smooth scroll-triggered animations.

Pages

	1.	Home Page:
	•	Introduction and call-to-action buttons to download the resume or connect.
	2.	Skills Page:
	•	Overview of technical and soft skills.
	3.	Project Portfolio:
	•	Highlighted projects with a GitHub button for exploring all my work.
	4.	Hire Me:
	•	Contact options for potential collaborations and opportunities.
